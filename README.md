# Catering Marketplace

Welcome to the Catering Marketplace project! This application serves as a platform for connecting customers with caterers, allowing users to explore various catering services, view menus, and make bookings.

## Features

- **User-Friendly Interface**: A modern and responsive design built with TailwindCSS.
- **Caterer Listings**: Browse through a variety of caterers with detailed information.
- **Search Functionality**: Easily find caterers based on categories and services.
- **Customer Reviews**: Read reviews from other customers to make informed decisions.

## Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites

- Node.js (version 14 or later)
- npm (Node Package Manager)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/catering-marketplace.git
   ```

2. Navigate to the project directory:

   ```bash
   cd catering-marketplace
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

### Running the Application

To start the development server, run:

```bash
npm run dev
```

Your application will be running at `http://localhost:3000`.

### Building for Production

To create an optimized production build, run:

```bash
npm run build
```

Then, you can start the production server with:

```bash
npm start
```

## Folder Structure

- `src/app`: Contains the main application files, including layout and pages.
- `src/components`: Contains reusable components like Header, Footer, and CaterCard.
- `src/lib`: Contains utility functions.
- `src/types`: Contains TypeScript interfaces and types.
- `public`: Contains static assets like images and icons.
