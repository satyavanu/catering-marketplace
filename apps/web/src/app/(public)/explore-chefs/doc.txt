# 5. 🔥 3 CUSTOMER FLOWS (MOST IMPORTANT)

You mentioned confusion — this is the core decision.

---

## FLOW 1: ⚡ Quick Order (Zomato-style)

### Best for:

* Daily food
* Small orders

### Flow:

```
Explore → Chef → Menu → Add to Cart → Checkout → Delivery
```

### Key Features:

* Instant checkout
* No negotiation
* Fixed pricing

---

## FLOW 2: 📩 Enquiry / Quote (UrbanClap-style)

### Best for:

* Events
* Catering
* Custom needs

### Flow:

```
Explore → Chef → Click "Enquire"

→ Form:
  - Event type
  - Date
  - Guests
  - Budget

→ Submit → Chef responds → User confirms → Pay advance
```

---

## FLOW 3: 🧑‍🍳 Direct Booking (Hybrid Premium Flow)

### Best for:

* Private chefs
* Meal plans

### Flow:

```
Explore → Chef → Select Package

→ Choose:
  - Date
  - Duration
  - Guests

→ Pay → Booking confirmed
```

---

# 6. 🧠 Recommended Strategy (VERY IMPORTANT)

Don’t force one flow.

### Instead:

| Chef Type    | Default Flow |
| ------------ | ------------ |
| Home Chef    | Quick Order  |
| Caterer      | Enquiry      |
| Private Chef | Booking      |
| Meal Plans   | Subscription |

---

## Smart CTA Logic

```ts
if (chef.type === "HOME_CHEF") show("Order Now")
if (chef.type === "CATERER") show("Get Quote")
if (chef.type === "PRIVATE_CHEF") show("Book Chef")
```
